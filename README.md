# AI-Powered Student Learning Management System

A machine learning-based system to detect struggling students early, predict exam scores, classify dropout risk, and generate personalized study plans.

## Features

- Isolation Forest anomaly detection
- XGBoost exam score prediction
- Random Forest dropout classification
- K-Means learning style clustering
- SHAP explainability
- Streamlit dashboard

## Tech Stack

- Python
- Scikit-Learn
- XGBoost
- SHAP
- Streamlit
